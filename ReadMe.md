Prototype Demonstration

A working prototype has been developed to visualize the proposed solution and user flow.

👉 View Prototype:
https://www.figma.com/proto/ki8k2goCRdQ69VDNmNhWk1/IELTS-Engagement-App

The prototype showcases gamified engagement features including point-based scoring, leaderboard rankings, competitive challenges, and placement-driven incentives.

This screen focuses on encouraging daily engagement through contests and performance tracking. It displays a “Today’s Contest” section where users can participate in a reading test and earn points within a limited time. The timer creates urgency, motivating users to take action immediately.Below this, users can view their personal performance, including number of contests played, accuracy percentage, and current rank. A graphical progress chart helps users visualize their improvement over time. This combination of real-time contests and performance tracking encourages consistency and builds competitive motivation.

![alt text](https://github.com/Shreya-Pathak-90/IELTS-Design-for-User-Engagement/blob/main/Screenshot%202026-03-29%20215241.png)

This screen highlights the leaderboard and integrates career incentives with performance. Users can view rankings of top-performing students along with their total points. The Top performers are associated with well-known companies, representing placement or internship opportunities.This creates a strong motivation for users to improve their rank, as higher performance directly connects to better career prospects. The search functionality allows users to compare their performance with others, fostering healthy competition and aspirational growth.

![alt text](https://github.com/Shreya-Pathak-90/IELTS-Design-for-User-Engagement/blob/main/Screenshot%202026-03-29%20215303.png)

This screen promotes peer interaction and competitive learning through challenges. Users can start playing immediately using the “Play Now” feature and see the number of active participants in real time, creating a sense of community.The challenges section includes different types of activities such as daily quizzes, speed-based trivia, and speaking improvement tasks. Each challenge shows the number of participants, encouraging users to join and compete. This feature enhances engagement by making learning interactive, social, and competitive.

![alt text](https://github.com/Shreya-Pathak-90/IELTS-Design-for-User-Engagement/blob/main/Screenshot%202026-03-29%20215322.png)

These screens collectively create a gamified learning environment where students are motivated through daily contests, peer competition, and career-oriented rewards, leading to consistent engagement and improved learning outcomes.
